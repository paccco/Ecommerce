Tarea 7: API Resfull

En esta tarea haremos una API Restful para la tabla de productos. Hablitaremos los siguientes endpoints: 

GET /api/productos          // todos los productos

GET /api/productos/id       // producto con id
	
PUT /api/producto/id        // cambiar el titúlo/descripción/precio de un producto

DELETE /api/productos/id    // eliminar un producto

POST /api/productos         // añadir un producto

Seguimos Building a REST API with Prisma and express.js, pero en la primera ruta (ver todos los productos) pondremos opciones para paginación y ordenación a traves de parámetros en el query, 

GET /api/productos?desde=1&hasta=20&ordenación=ascendente

Testing

Utillizaremos la extensión de VSCode REST Client para la comprobación y depuración del API. Los endpoints los pondremos en un archivo test-api.http para utilizarlos desde REST Client:
