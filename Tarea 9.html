Tarea 9: SPA con Vite, React y Tailwind

En esta tarea haremos un pequeño front con dos componentes React. Uno mostrará una imagen desde una api externa, y el otro desde el api de la tienda.

Empezaremos también a utilizar Tailwind CSS, que veremos mas a fondo en el resto del curso.

Utilizaremos Vite como servidor de desarrollo.

Instalación

Instalamos primero Vite escogiendo las opciones de TypeScript y React. A continuación añadimos Tailwind como pone en Get started with Tailwind CSS. Para instalar tipos de letra de google en tailwind, se cambia index.css: 

@import url("https://fonts.googleapis.com/css2?family=Montserrat:wght@400;700&family=EB+Garamond:ital@0;1&display=swap");
@import "tailwindcss";
	
@theme {
  --font-montserrat: "Montserrat", sans-serif; 
};
y ya se puede utilizar p.e.: 
...
<h3 className="font-montserrat"> Galeria de Tienda Prado </h3>
...

CORS

Previamente prepararemos otro end point que sirva imágenes aleatorias de desde el serivor de la tienda. El API no funcionará debido a que está servido desde un puerto del vite, distinto del de servidor. Estas llamadas no se aceptan debido al CORS. Para poder utilizarlo durante el desarrollo habrá que configurarlo en el servidor.

App.tsx

El componente principal queda: 

 // App.tsx
import Perritos from './components/Perritos.tsx'
import Cuadros from './components/Cuadros.tsx'

function App() {
  return (
    <div className="flex gap-2 items-center justify-center h-screen font-montserrat">
      <Perritos />
      <Cuadros />
    </div>
  )
}
export default App
